# Competitor Matrix

*Extracted from MoltPe Research reports, March 2026. Covers agent-native payment infrastructure players.*

## Agent Collections / Receivables

| Company | Product | Capability | Limitation |
|---------|---------|------------|------------|
| **PayPal** | MCP server | Invoice creation, sending, reminders, QR codes, subscription management, refunds, disputes, financial reporting — all via natural language | Ecosystem-locked: requires PayPal accounts on both sides |

*Note: PayPal MCP is the only existing solution with meaningful receivables functionality. All other infrastructure focuses on agents-as-buyers.*

## Competitive Gap Summary

| Gap | Competitors in space |
|-----|---------------------|
| Agent receivables / collections | PayPal MCP only (ecosystem-locked) |
| Emerging market agent payments (stablecoin-native) | Zero |
| Stablecoin-to-local-currency off-ramps for agents | Zero agent-native |
| Agent dispute resolution | Zero |
| Multi-currency agent payments | Minimal (USDC-only in practice) |
| Agent KYC for regulated emerging markets | Zero (Skyfire KYA is US-centric) |
