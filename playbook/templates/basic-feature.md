# Basic Feature Template

A single-sprint template for straightforward feature work.

## Structure

```
Sprint: [Feature Name]
Duration: 1 session
Agent count: 1
```

## Checklist

1. [ ] Read requirements from plan file
2. [ ] Create or modify files per spec
3. [ ] Run lint and format checks
4. [ ] Run tests
5. [ ] Update plan file status
6. [ ] Commit with descriptive message

## Commit Message Format

```
Built: [what was built]
Tested: [what was tested]
Status: Pass
```

## When to Use

- Single feature with no parallel dependencies
- Bug fixes
- Documentation updates
- Configuration changes
